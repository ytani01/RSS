# RSSリーダー


## usage

$ rss.py url


### ex.

$ rss.py https://rss-weather.yahoo.co.jp/rss/days/4610.xml


## install

```
$ cd
$ python3 -m venv env-rss
$ . ~/env-rss/bin/activate
(env-rss)$ cd ~/env-rss
(env-rss)$ git clone https://github.com/ytani01/RSS.git
(env-rss)$ cd RSS
(env-rss)$ pip install -r requirements.txt
```
